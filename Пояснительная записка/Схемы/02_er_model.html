ER-модель базы данных

Модель отражает связи между пользователями, музыкальным каталогом, рецензиями, оценками, лайками и предпочтениями профиля.

PostgreSQL
users
  • PK id
  • username, email
  • password_hash
  • profile fields
artists
  • PK id
  • name
  • image_url
  • verified
albums
  • PK id
  • FK artist_id
  • FK genre_id
  • cover_url
tracks
  • PK id
  • FK album_id
  • duration
  • track_number
subscriptions
  • PK id
  • FK follower_id
  • FK author_id
  • created_at
reviews
  • PK id
  • FK user_id
  • FK album_id / track_id
  • scores, status
review_likes
  • PK id
  • FK user_id
  • FK review_id
  • created_at
preferences
  • PK id
  • FK user_id
  • entity_type
  • entity_id, position
